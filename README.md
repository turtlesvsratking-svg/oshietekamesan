# oshietekamesan
# 看護研究サポートツール 🐢 教えてカメさん｜検索ツール

看護師や研究初心者のための、完全無料・インストール不要の軽量Webアプリ版ナビゲーションツールです。
情報メディア「[教えてカメさん](https://kamesan-kamesan.com/)」のコンテンツと密接に連動し、臨床現場や自宅での「困った！」の解決を素早く直感的にサポートします。

スマートフォンやタブレットの「ホーム画面に追加（ブックマーク）」機能を使った際、かわいいカスタムアイコン（`IMG_4323.png`）がスマホの画面上にきれいに表示されるウェブクリップ設定済み（PWA対応仕様）です。

## 🌟 主な機能

1. **看護研究ワード検索**
   * 看護研究の最重要用語（PICO、質的研究、概念枠組みなど）の意味をクイック検索。そのままブログの該当解説記事へとスムーズにナビゲートします。
2. **研究デザインの選択**
   * 臨床における研究目的や「明らかにしたいこと」に合わせて、5大デザイン（質的研究 / 量的記述研究 / 量的相関研究 / 準実験研究 / 混合研究法）から、最適な方法の選択をサポートします。
3. **統計解析方法の選択**
   * 扱うデータの種類（カテゴリーデータ / 連続変数）や比較したいグループの数（2群 / 3群以上）といった研究デザインに適合した統計手法の選択をサポートします。
